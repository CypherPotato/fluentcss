Acrylic & Mica

A translucent material recommended for panel backgrounds.

Acrylic panes

Acrylic panel
Acrylic panel with pos-z 1 shadow

Acrylic border radius is optional. The base component win-acrylic doens't carry border-radius, size or other layout properties.

You must include an reference to the acrylic mask PNG in main.css. 


                    
                        <div class="win-acrylic">
                            Acrylic panel
                        </div>

Toggle Mica effects

This is an mica panel. Toggle it to make it active. 


                    
                        <div class="win-mica focusable" id="mica-example">
                            <p>
                                This is an mica panel. Toggle it to make it active.
                            </p>
                            <script>
                                function toggleMica() {
                                    document.getElementById("mica-example").classList.toggle("active");
                                }
                            </script>
                            <button onclick="toggleMica()" class="win-button">
                                Toggle Mica state
                            </button>
                        </div>
