Context menu

Contexts menus are windows with contextual options to what the user is doing, being able to decide to do something with an item, object or action.

Simple context menu

Right click anywhere to open the context menu 


                    
                        <script>
                            document.querySelector("#viewboxContainer")
                                .addEventListener("contextmenu", () => {
                                    win.contextMenu([
                                        {
                                            onclick: () => {
                                                alert("hello");
                                            },
                                            text: "Show alert",
                                            img: "/etc/icons/star.svg"
                                        },
                                        {
                                            enabled: false,
                                            onclick: () => {
                                                alert("hello");
                                            },
                                            text: "Disabled item"
                                        }
                                    ]);
                                });
                        </script>
                        <p>
                            Right click anywhere to open the context menu
                        </p>
